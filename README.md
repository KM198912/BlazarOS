# BlazarOS
BlazarOS is a hobby operating system built for the x86_64 architecture. Here are its features:

- [x] GDT/IDT Tables
- [x] MM
    - [x] PMM
    - [x] VMM
    - [x] Heap
- [x] SMP
    - [x] ACPI
    - [x] MADT
    - [x] APIC/LAPIC/IOAPIC
    - [x] Multi-Tasking
        - [ ] IPC
- [x] Dev
    - [x] PS/2
        - [x] Keyboard
        - [ ] Mouse
    - [x] Serial
    - [x] ATA
    - [x] PIT/HPET
    - [ ] PCI
    - [ ] RTC Clock
    - [ ] SB16 Driver
    - [ ] Internet
    - [ ] RTL-8139 Driver
        - [ ] TCP/IP Stack
        - [ ] ARP
        - [ ] UDP
        - [ ] IPV4
        - [ ] DHCP
        - [ ] DNS
        - [ ] HTTP
- [x] FAT32 Driver
    - [x] Read operations
    - [x] Write operations
    - [x] Modify operations
    - [ ] Delete operations
- [x] VFS
- [ ] ELF
    - [ ] Syscalls
- [ ] GUI
    - [ ] WM
        - [ ] Window compositor
        - [ ] Elements
- [ ] Custom toolchain
